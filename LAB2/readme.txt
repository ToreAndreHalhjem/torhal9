Kontrollspørsmål lab 2:

Min versjon av nettsiden baserer seg på MVC(Model-View-Controller). MVC gjør det lettere å ha kontroll på nettsiden som skal vises.
HTML filen min er modellen her. Den holder på innholdet som skal vises.
CSS filen er "vieweren". Den bestemmer hvordan innholdet på siden skal vises.
Javascript-filen er den som kontrollerer hva som er vises av innhold i HTML-filen.
